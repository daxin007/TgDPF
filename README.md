# 文件介绍

<br><br>

- `/doc`
  项目相关文档
- `/log`
  保存试验记录文件
- `/record`
  保存实验结果
- `/synthetic_wind_speed`
  天气预报数据
- `/江苏十分钟数据`
  训练数据 
- `/tools`
  工具脚本，用于统计和比较数据、生成批处理文件、实验总结等
- `/tst`
  Transformer模型定义
- `/paper_writing` 
  写作相关文档
- `train_clean.py`  
- 模型训练
- `dataset.py`
  数据预处理和训练测试数据生成
- `model.py`
  模型定义
- `util.py`
  工具函数定义
- `configs.py` 
  超参数定义
